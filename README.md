# PixelOS

A pixel-meets-glass WebOS built for Hack Club Stardance. Pure HTML, CSS, and JavaScript — zero dependencies.

## Features

- Draggable, resizable windows with glassmorphism styling
- Wikipedia API viewer (search and read articles in-app)
- Pixel Art Editor (16×16 grid, flood fill, PNG export)
- Calculator
- Lo-Fi Cafe (direct audio streams from SomaFM)
- Glass Diary (write and export notes)
- Desktop widgets: analog clock + live calendar
- Day/night theme with pixel rain
- Fully customizable settings (accent color, effects, glass blur)

## Usage

Open `index.html` in any browser. No build step, no install.
